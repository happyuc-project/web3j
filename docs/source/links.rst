Links and Useful Resources
==========================

- HappyUC `Homestead Documentation <https://happyuc-homestead.readthedocs.io/en/latest/>`_
- HappyUC `Wiki <https://github.com/happyuc-project/wiki/wiki>`_
- HappyUC `JSON-RPC specification <https://github.com/happyuc-project/wiki/wiki/JSON-RPC>`_
- HappyUC `Yellow Paper <gavwood.com/paper.pdf>`_ and
  `GitHub <https://github.com/happyuc-project/yellowpaper>`_ repository
- `Homestead <https://happyuc-homestead.readthedocs.org/en/latest/>`_ docs
- `Solidity <http://solidity.readthedocs.io/en/develop/>`_ docs
- `Layout of variables in storage <http://solidity.readthedocs.io/en/latest/miscellaneous.html#layout-of-state-variables-in-storage>`_
- `HappyUC tests <https://github.com/happyuc-project/tests>`_ contains lots of common tests for clients
- `Hucscan <https://hucscan.io>`_ is very useful for exploring blocks and transactions, it also
  has a `testnet site <https://testnet.hucscan.io>`_
- `Hucstats <https://ethstats.net/>`_ provides a useful network dashboard. There is also a
  dedicated `Parity dashboard <https://stats.parity.io/>`_,
  `Rinkeby testnet dashboard <http://rinkeby.io/>`_, and one for the
  `Kovan testnet <http://kovan-stats.parity.io/>`_.
- `HappyUC reddit <https://www.reddit.com/r/happyuc/>`_
